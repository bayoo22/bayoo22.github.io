Vue-router的安装与使用

什么是vue-router？

路由，其实就是指向、映射的意思。

基本概念

　路由中有三个基本的概念 route, routes, router。

  1. route 是一条路由Home按钮 => home内容

  2. routes 是一组路由，把上面的每一条路由组合起来，形成一个数组。
  3. router 是一个机制，相当于管理者，它来管理路由。当用户点击Home 按钮的时候，这时router 就起作用了，它到routes 中去查找，去找到对应的 Home 内容，所以页面中就显示Home 内容。
  4. 客户端中的路由 是dom元素的显示和隐藏。客户端路由有两种实现方式:基于hash 和基于html5 history api.

vue-router安装

npm install vue-router

vue-router使用

需要处理:

1. 处理页面点击的部分;
2. 点击之后需要到达的路径。

1.页面实现(html模板)

<router-link to="/home">Home</router-link>

<router-view></router-view>

2. 配置路由(JS)

  1. 定义一条路由,该路由对象由两个部分组成：
    path和component，
    如{path:'/home',component:home;}
  2. 定义routers组
    const routes = { {path: '/home',component:Home}, {path:'/about',component:About } }
  3. 创建router对路由的管理,由 new vueRouter()创建，接受routers 参数。
    const router = new VueRouter({routers)}
  4. 将实例化的router，放入vue的根实例中
    const app = new Vue({ router }).$mount('#app')
    完成后就可以使用路由功能了。
